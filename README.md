# TerrorSquadAI — v9.3.0 [God-Tier] ⚔️🛡️

IA de mascotas táctica. Tus demonios dejan de ser simples esbirros para convertirse en combatientes inteligentes.

## 🚀 Características
- Optimización extrema para Turtle WoW.
- Integración nativa con el **WCS_Brain**.
- Interfaz modernizada y limpia.

## ⚙️ Instalación
1. Descarga el ZIP.
2. Extrae en Interface\AddOns\TerrorSquadAI.
3. Reinicia el juego.
