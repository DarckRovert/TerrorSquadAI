# TerrorSquadAI (La Mente Maestra) - Séquito Edition v7.0

![banner táctico](https://raw.githubusercontent.com/DarckRovert/TerrorSquadAI/master/media/banner.png)

### 🌐 Vínculo Maestro del Ecosistema
Este addon es el nexo central de la red táctica de 10 addons del clan. Coordina la telemetría de combate, economía y navegación para una dominación total.

## 🚀 Módulos Principales

### 1. El Cerebro (Macros Dinámicas)
Permite crear una macro inteligente que cambia su función según el contexto.
*   **Comando**: `/tsai macro`
*   **Uso**: Crea una macro llamada `TSA_Brain`.

### 2. El Radar Táctico 2.0
Visualización de enemigos y aliados en un radar holográfico.

### 3. Séquito Ecosystem (Bridges)
Detección automática de los otros 9 addons del clan (WCS_Brain, pfUI, etc.).

## ⚡ Comandos Rápidos
*   `/tsai` - Abre el panel central.
*   `/tsai config` - Configuración avanzada.
*   `/tsai status` - Estado de la red Séquito.
